# Club

Voluntary membership organizations -- social clubs, sports leagues, hobby groups, fraternal orders -- that set rules for members through bylaws and informal norms. Entry is voluntary and exit costs are low; you can quit at any time. Scope is narrow, limited to club activities. Structure and formality vary between written constitutions and unwritten customs.

## Dimensional Profile

| Dimension | Level |
|:--|:--|
| Penalty severity | Low |
| Cost of exit | Low |
| Entry | Voluntary |
| Scope | Narrow |
| Structure | Mixed |
| Formality | Mixed |

## Typical Rules
- Dress code for club events or facilities
- Mandatory attendance at a minimum number of meetings per year to retain membership
- Prohibition on recruiting members from rival organizations
